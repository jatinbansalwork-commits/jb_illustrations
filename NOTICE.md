# Notice

JB Illustrations is an AI-agent skill adapted from Ian Xiaohei Illustrations.

The bundled example images in `examples/images/` and `jb_illustrations/assets/examples/` were originally generated by Ian as style calibration examples for the source skill, and are included here under the same MIT license for visual reference.

The recurring character **JB** is a reimagined visual IP for English inline illustrations with an Indian cultural lens. When redistributing or adapting this repository, please credit the original project:
